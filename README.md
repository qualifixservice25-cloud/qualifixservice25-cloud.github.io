# Qualifix Service - Calcolo Cartongesso

Repository ufficiale per l'hosting della landing page e dei file di configurazione dell'app **Calcolo Cartongesso**.

## Contenuti del Repository
- `index.html`: Landing page ufficiale dell'applicazione.
- `privacy-policy.html`: Informativa sulla privacy richiesta da Google Play.
- `app-ads.txt`: File di verifica per Google AdMob (IAB Tech Lab standard).
- `.nojekyll`: File di configurazione per permettere la scansione dei file tecnici.
